# Screenshots

Place your app screenshots here. Recommended captures:

1. **chat-list.png** — Main screen showing recent conversations
2. **messages.png** — Open chat with message bubbles
3. **challenges.png** — Challenge cards (both active and completed)
4. **profile.png** — User profile with settings
5. **suggestions.png** — "Suggest Me" screen with compatibility percentage
6. **search.png** — Search/filter screen

## How to capture

On iOS Simulator: `Cmd + S`
On Android Emulator: `Cmd + Shift + S` (or the camera button in the toolbar)
On physical device: Use Expo DevTools → "Take Screenshot"
